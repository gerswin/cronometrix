<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → /api/v1/leaves (write) | Authenticated admin JWT holder posting multipart form including evidence file. |
| HTTP client → /api/v1/leaves (read) | Any authenticated JWT holder (3 roles) per Phase 1 D-09. |
| Multipart upload → filesystem | Evidence file bytes written under `./data/leaves/{leave_id}.{ext}`. Path is server-generated from UUID — never user-controlled. |
| Engine ← leaves table | recompute_for_day queries leaves via parameterized SQL; leave row becomes EngineInput.leave. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-3-14 | Tampering | Admin creates overlapping leave rows to zero out an employee's records fraudulently | mitigate | `leaves::service::create_leave` rejects with `LeaveConflict` if an overlapping active leave exists (`from_date <= ?new_to AND to_date >= ?new_from AND status='active' AND deleted_at IS NULL`). Audit trigger records actor via app-code secondary audit insert. |
| T-3-15 | Tampering | Evidence path traversal via `..` or absolute path | mitigate | Evidence path is SERVER-GENERATED: `format!("{}.{}", Uuid::new_v4(), ext)` under `./data/leaves/`. User-supplied filename is discarded. Read path uses canonicalize + starts_with check (mirror events/handlers.rs `get_event_photo`). |
| T-3-16 | Tampering | Non-image/non-PDF evidence upload (malware executables) | mitigate | validator on `CreateLeaveMultipart`: content_type must be `application/pdf` or `image/jpeg` or `image/png`. File size limit 10MB. Reject in service layer before writing to disk. |
| T-3-17 | Repudiation | Leave created/cancelled without actor audit | mitigate | Migration 011 installs AFTER INSERT/UPDATE/DELETE triggers on `leaves` that insert audit_log rows (old_data/new_data JSON, actor_id=NULL in trigger; app-code sets a secondary audit row via a Phase-2-style `command_audit_log` helper when actor context is needed). |
| T-3-18 | Information Disclosure | Unauthenticated access to evidence files | mitigate | Evidence served via `GET /api/v1/leaves/{id}/evidence` behind `require_auth` (mirror events/handlers.rs `get_event_photo`). Path canonical + under `./data/leaves` check. |
| T-3-19 | Elevation of Privilege | Supervisor/viewer creates or cancels leave | mitigate | POST /api/v1/leaves + DELETE /api/v1/leaves/{id} mounted behind `require_admin` middleware. Verified by integration test `create_leave_forbidden_for_supervisor`. |
| T-3-20 | Tampering | SQL injection via `leave_type`, `employee_id`, `justification` fields | mitigate | All DB writes use libsql::params! positional binding. `leave_type` validated against enum via validator::Validate + CHECK constraint on column. |
| T-3-21 | Denial of Service | Large evidence file consumes memory during multipart parse | mitigate | axum Multipart iterator streams chunks; service layer reads fully into Vec<u8> with 10MB cap (return 422 if exceeded before full read). |

-- Leave records per D-13. Full-day only (D-14). Immediate approval (D-15). Overlay precedence (D-16).
-- Soft-delete via status+deleted_at; never hard-delete (Venezuelan LOTTT legal retention).

CREATE TABLE IF NOT EXISTS leaves (

    from_date TEXT NOT NULL,             -- 'YYYY-MM-DD' inclusive
    to_date TEXT NOT NULL,               -- 'YYYY-MM-DD' inclusive

    justification TEXT NOT NULL,
    evidence_path TEXT,
    created_by TEXT NOT NULL REFERENCES users(id),
    status TEXT NOT NULL DEFAULT 'active' CHECK(status IN ('active', 'cancelled')),
    deleted_at INTEGER,
    version INTEGER NOT NULL DEFAULT 1,
    created_at INTEGER NOT NULL,
    updated_at INTEGER NOT NULL
);

CREATE INDEX IF NOT EXISTS idx_leaves_employee ON leaves(employee_id);
CREATE INDEX IF NOT EXISTS idx_leaves_dates ON leaves(from_date, to_date);

pub async fn create_leave(

) -> Result<(StatusCode, Json<LeaveResponse>), AppError> {

    let mut employee_id: Option<String> = None;
    let mut from_date: Option<String> = None;
    let mut to_date: Option<String> = None;
    let mut leave_type: Option<String> = None;
    let mut justification: Option<String> = None;
    let mut evidence_bytes: Option<Vec<u8>> = None;
    let mut evidence_ext: Option<&'static str> = None;

    while let Some(field) = multipart.next_field().await.map_err(|e| AppError::Validation {
        code: "VALIDATION_ERROR",
        message: format!("malformed multipart: {}", e),
    })? {

        match name.as_str() {
            "employee_id" => employee_id = Some(field.text().await.map_err(|e| AppError::Validation {
                code: "VALIDATION_ERROR", message: e.to_string(),
            })?),

                code: "VALIDATION_ERROR", message: e.to_string(),
            })?),
            "to_date" => to_date = Some(field.text().await.map_err(|e| AppError::Validation {
                code: "VALIDATION_ERROR", message: e.to_string(),
            })?),
            "leave_type" => leave_type = Some(field.text().await.map_err(|e| AppError::Validation {
                code: "VALIDATION_ERROR", message: e.to_string(),
            })?),
            "justification" => justification = Some(field.text().await.map_err(|e| AppError::Validation {
                code: "VALIDATION_ERROR", message: e.to_string(),
            })?),
            "evidence" => {
                let ct = field.content_type().unwrap_or("").to_string();
                evidence_ext = match ct.as_str() {
                    "application/pdf" => Some("pdf"),
                    "image/jpeg" => Some("jpg"),

                        code: "VALIDATION_ERROR",
                        message: format!("evidence content_type must be application/pdf, image/jpeg, or image/png (got '{}')", ct),
                    }),
                };

                    code: "VALIDATION_ERROR",
                    message: format!("reading evidence bytes: {}", e),
                })?;

                        code: "VALIDATION_ERROR",
                        message: format!("evidence file exceeds 10MB (got {} bytes)", bytes.len()),
                    });

            }
            _ => {

                let _ = field.bytes().await;
            }
        }

        // Simpler: persist to service with None evidence_relpath, get back the leave id, then rename.

        let relpath = format!("{}.{}", leave_id_preview, ext);
        write_photo_atomic(&service::leaves_root(), &relpath, bytes).map_err(AppError::Internal)?;
        Some(relpath)
    } else { None };

    // 3. Call service.
    let req = CreateLeaveRequest { employee_id: employee_id.clone(), from_date: from_date.clone(), to_date: to_date.clone(), leave_type, justification };
    let leave = service::create_leave(
        &state.db.connect().map_err(|e| AppError::Internal(e.into()))?,
        &claims.sub,
        req,
        evidence_relpath,
    ).await?;

    if let Some(tx) = state.recompute_tx.as_ref() {

                employee_id: employee_id.clone(),
                anchor_date: d,
            });
            d = d.succ_opt().unwrap();
        }
    }

}

pub async fn cancel_leave(
    State(state): State<AppState>,
    Path(id): Path<String>,
    Query(q): Query<CancelQuery>,
) -> Result<StatusCode, AppError> {
    let conn = state.db.connect().map_err(|e| AppError::Internal(e.into()))?;
    // Fetch employee_id + date range BEFORE cancelling so we can recompute.
    let leave = service::get_by_id(&conn, &id).await?;
    service::cancel(&conn, &id, q.version).await?;

    if let Some(tx) = state.recompute_tx.as_ref() {
        let from = NaiveDate::parse_from_str(&leave.from_date, "%Y-%m-%d").unwrap();
        let to = NaiveDate::parse_from_str(&leave.to_date, "%Y-%m-%d").unwrap();
        let mut d = from;
        while d <= to {
            let _ = tx.send(RecomputeRequest {
                employee_id: leave.employee_id.clone(),
                anchor_date: d,
            });
            d = d.succ_opt().unwrap();
        }
    }

    Ok(StatusCode::NO_CONTENT)
}

pub async fn get_leave_evidence(
    State(state): State<AppState>,
    Path(id): Path<String>,
) -> Result<Response, AppError> {
    let conn = state.db.connect().map_err(|e| AppError::Internal(e.into()))?;
    let leave = service::get_by_id(&conn, &id).await?;
    let relpath = leave.evidence_path.ok_or_else(|| AppError::NotFound {
        code: "LEAVE_EVIDENCE_NOT_FOUND",
        message: "Leave has no evidence attached".into(),
    })?;

    if relpath.contains("..") || relpath.starts_with('/') {
        return Err(AppError::NotFound { code: "LEAVE_EVIDENCE_NOT_FOUND", message: "Evidence not available".into() });
    }

    let root = service::leaves_root();
    let root_canonical = root.canonicalize().map_err(|_| AppError::NotFound {
        code: "LEAVE_EVIDENCE_NOT_FOUND", message: "Evidence not available".into(),
    })?;
    let full = root_canonical.join(&relpath);
    let canonical = full.canonicalize().map_err(|_| AppError::NotFound {
        code: "LEAVE_EVIDENCE_NOT_FOUND", message: "Evidence not found on disk".into(),
    })?;
    if !canonical.starts_with(&root_canonical) {
        return Err(AppError::NotFound { code: "LEAVE_EVIDENCE_NOT_FOUND", message: "Evidence not available".into() });
    }

    let bytes = tokio::fs::read(&canonical).await.map_err(|_| AppError::NotFound {
        code: "LEAVE_EVIDENCE_NOT_FOUND", message: "Evidence not found on disk".into(),
    })?;

    let ct = match PathBuf::from(&relpath).extension().and_then(|s| s.to_str()) {
        Some("pdf") => "application/pdf",
        Some("jpg") | Some("jpeg") => "image/jpeg",
        Some("png") => "image/png",
        _ => "application/octet-stream",
    };
    let mut headers = HeaderMap::new();
    headers.insert(header::CONTENT_TYPE, HeaderValue::from_static(match ct {
        "application/pdf" => "application/pdf",
        "image/jpeg" => "image/jpeg",
        "image/png" => "image/png",
        _ => "application/octet-stream",
    }));
    Ok((StatusCode::OK, headers, bytes).into_response())
}
