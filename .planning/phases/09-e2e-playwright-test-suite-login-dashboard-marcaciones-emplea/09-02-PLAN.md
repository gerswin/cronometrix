<objective>
Lock the D-13 license-bypass safety contract: a test-only flag (`CRONOMETRIX_LICENSE_BYPASS=true`) is honored ONLY when `CRONOMETRIX_E2E=true` is also set; in any other configuration the bypass flag is treated as a sign of misconfiguration and the binary aborts with exit code 2 BEFORE entering the license check path. This prevents the test-only flag from leaking to a production deploy and silently disabling the hardware-bound license gate (LIC-05).

Purpose: Audit-compliance non-negotiable per CLAUDE.md and CONTEXT D-13. Without this safety, an operator who sourced a `.env.test` and forgot to clean up could ship a license-disabled production build. The integration test makes the safety unforgettable.
Output: Pure `evaluate_bypass(e2e: bool, bypass: bool) -> BypassDecision` function in `license::service`, wiring in `main.rs` that branches on its result, and an end-to-end integration test that spawns the real binary and verifies exit code 2.

This is the highest-risk plan in Phase 9 (per 09-RESEARCH.md). Implement TDD: tests for `evaluate_bypass` (RED), implementation (GREEN), then the binary spawn integration test, then the main.rs wiring.
</objective>

```rust
// backend/src/license/service.rs (NEW additions; preserve all existing code)

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum BypassDecision {
    /// Both flags set: skip license validation, mark license_valid=true.
    AllowBypass,
    /// Bypass set without E2E: abort startup with exit code 2.
    AbortMisconfigured,
    /// Neither set, or only E2E set: proceed to normal load_and_validate_license path.
    NormalPath,
}

/// Pure logic — no side effects, no env reads. Caller passes parsed booleans.
/// Locked by `tests/license_bypass_safety.rs` AND by inline unit tests below.
pub fn evaluate_bypass(e2e: bool, bypass: bool) -> BypassDecision {
    match (e2e, bypass) {
        (true, true)   => BypassDecision::AllowBypass,
        (false, true)  => BypassDecision::AbortMisconfigured,
        (true, false)  => BypassDecision::NormalPath,
        (false, false) => BypassDecision::NormalPath,
    }
}
```
